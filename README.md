# Информация за Норвегия

Този проект показва как да извлечете и визуализирате информация за Норвегия чрез публично API, използвайки JavaScript и Fetch API.

## Как да публикувате сайта с GitHub Pages

1. **Създайте нов публичен репозиторий** в GitHub (например `norway-info`).
2. **Качете файловете** `index.html` и `script.js` в основната директория на репозитория.
3. **Добавете, комитнете и пушнете файловете** чрез следните git команди:

```bash
git init
git remote add origin https://github.com/SvetoslavGochev/norway-info.git
git add .
git commit -m "Начален комит: Информация за Норвегия"
git branch -M main
git push -u origin main
```

4. **Активирайте GitHub Pages**:
   - Отидете в Settings → Pages на вашия репозиторий.
   - Изберете Branch: `main` и папка: `/ (root)`.
   - Запазете настройките.

5. **Достъп до сайта**:
   - След няколко минути сайтът ще бъде достъпен на адрес:
     `https://SvetoslavGochev.github.io/norway-info/`
